    - Email failures: catch exception, append error message to errors[], continue loop (never abort)
    - success=True if run() completes without raising an uncaught exception (partial email failures are normal)
    - success=False only if generate() raises or another fatal exception occurs; in that case, append error, return PipelineResult with success=False
    - records_processed counts PDFs found (not emails sent)
    - emails_sent counts successful send_comprehensive_report_email() calls that returned True
    - Use Python logging module (not print) for per-student lines: "[{report_type}] Sent: {email}" or "[{report_type}] Failed: {email}: {reason}"
    - At end of run(), log final summary: records_processed, emails_sent, len(errors)
  </behavior>
  <action>
    Create core/runner.py with the following structure:

    IMPORTS:
    ```python
    import logging
    import os
    from pathlib import Path
    from typing import List, Optional, TypedDict

    from reports import get_generator
    from core.email_sender import EmailSender
    from core.drive_service import DriveService

    logger = logging.getLogger(__name__)
    ```

    PIPELINERESULT TYPEDDICT:
    ```python
    class PipelineResult(TypedDict):
        success: bool
        records_processed: int
        emails_sent: int
        errors: List[str]
    ```

    PIPELINERUNNER CLASS:
    - __init__(self, report_type: str, dry_run: bool = False, test_email: Optional[str] = None)
      Store all three as self.report_type, self.dry_run, self.test_email

    - _extract_email_from_pdf(self, pdf_path: Path) -> Optional[str]
      Parse stem of filename (informe_{email}_{atype}).
      Split stem by '_'. If fewer than 3 parts or parts[0] != 'informe', return None.
      Email = '_'.join(parts[1:-1]). Return email or None if empty.

    - _send_email(self, recipient: str, pdf_path: Path, drive_link: Optional[str]) -> bool
      sender = EmailSender()
      return sender.send_comprehensive_report_email(
          recipient_email=recipient,
          pdf_content=pdf_path.read_bytes(),
          username=recipient,
          filename=pdf_path.name,
          drive_link=drive_link,
      )

    - _upload_to_drive(self, pdf_path: Path) -> Optional[str]
      try: return DriveService().upload_file(pdf_path, folder_id=os.getenv("GOOGLE_DRIVE_FOLDER_ID", ""))
      except Exception as exc: logger.warning(f"[{self.report_type}] Drive upload failed: {exc}"); return None

    - run(self) -> PipelineResult
      records_processed = 0
      emails_sent = 0
      errors: List[str] = []

      try:
          GeneratorClass = get_generator(self.report_type)
          generator = GeneratorClass()
          output_path = generator.generate()
      except Exception as exc:
          errors.append(str(exc))
          return PipelineResult(success=False, records_processed=0, emails_sent=0, errors=errors)

      # Collect PDFs
      if output_path.is_dir():
          pdfs = sorted(output_path.glob("*.pdf"))
      else:
          pdfs = [output_path] if output_path.suffix == ".pdf" else []

      records_processed = len(pdfs)

      for pdf_path in pdfs:
          student_email = self._extract_email_from_pdf(pdf_path)
          if not student_email:
              logger.warning(f"[{self.report_type}] Could not extract email from {pdf_path.name}, skipping")
              errors.append(f"Could not extract email from {pdf_path.name}")
              continue

          if self.dry_run:
              logger.info(f"[{self.report_type}] Dry-run: would send to {student_email} ({pdf_path.name})")
              continue

          recipient = self.test_email if self.test_email else student_email
          drive_link = None

          if not self.test_email:
              drive_link = self._upload_to_drive(pdf_path)

          try:
              sent = self._send_email(recipient, pdf_path, drive_link)
              if sent:
                  emails_sent += 1
                  logger.info(f"[{self.report_type}] Sent: {student_email} -> {recipient} ({pdf_path.name})")
              else:
                  errors.append(f"Email returned False for {student_email}")
                  logger.warning(f"[{self.report_type}] Failed: {student_email}: send returned False")
          except Exception as exc:
              errors.append(f"Email error for {student_email}: {exc}")
              logger.error(f"[{self.report_type}] Failed: {student_email}: {exc}")

      logger.info(
          f"[{self.report_type}] Pipeline complete: "
          f"records_processed={records_processed} emails_sent={emails_sent} errors={len(errors)}"
      )
      return PipelineResult(
          success=True,
          records_processed=records_processed,
          emails_sent=emails_sent,
          errors=errors,
      )
    ```

    Python 3.9 compatibility note: TypedDict from typing works in 3.9. Use Optional[str] not str | None in signatures.
